# 更新日志 (Changelog)

所有值得注意的版本变化都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-03-23

### 🎉 正式发布

首个正式版本，包含完整的体重追踪功能。

### ✨ 新增功能

#### 核心功能
- 体重记录功能（日期、时间、体重、心情、备注）
- 趋势分析页面（支持周/月/年视图）
- 折线图展示体重变化趋势
- BMI 自动计算与健康建议
- 目标进度追踪（起始体重、目标体重）
- 历史数据列表（按月度聚合）
- 记录详情查看、编辑、删除

#### 数据管理
- Excel 导出（支持自定义保存路径）
- Excel 导入（增量覆盖模式）
- JSON 备份导出
- 从备份恢复数据
- 自动清理 1 年前旧数据
- 数据迁移功能

#### 界面特性
- Material Design 3 设计语言
- 刘海屏/挖孔屏适配
- Edge-to-Edge 全屏显示
- 深色模式支持

### 🛠️ 技术实现
- Jetpack Compose UI 框架
- MVVM + Clean Architecture 架构
- Hilt 依赖注入
- Room 数据库
- Kotlin Coroutines + Flow
- 单元测试 + 集成测试
